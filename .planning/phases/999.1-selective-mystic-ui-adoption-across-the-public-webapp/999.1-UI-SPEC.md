## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Tight inline gaps, small badge internals, compact source metadata spacing |
| sm | 8px | Chip gaps, wrapped link groups, compact proof-row spacing |
| md | 16px | Default panel/card internals, grid gaps, route model spacing |
| lg | 24px | Section child spacing, support panels, comparison/proof lane padding |
| xl | 32px | Larger reading-surface padding and route group spacing |
| 2xl | 48px | Major section breaks within the existing `PageSection` rhythm |
| 3xl | 64px | Page-level spacing only; do not add decorative page bands |

Exceptions: Preserve existing 20px `p-5`, `gap-5`, and `mt-5` where already used by established editorial cards and route panels. Preserve existing radii `--radius-card: 16px` and `--radius-soft: 24px`. Existing `data-chip` controls must keep at least 40px height. If any icon-only control is introduced, it must be at least 44px square, but this phase should not add icon-only controls.

New phase UI must not nest cards inside cards. Use full-width route sections with constrained inner content, or a single `panel-wash`/`support-panel` where a real proof or comparison grouping needs a frame.

## Typography

Use exactly these four type roles for new Phase 999.1 UI. Preserve existing component typography when no defect exists.

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Label / metadata | 12px | semibold 600 | 1.4 |
| Supporting UI text | 14px | regular 400 | 1.5 |
| Body / route lead | 16px | regular 400 | 1.5 |
| Section heading | 30px | semibold 600 | 1.2 |

Weights are limited to regular 400 and semibold 600. New headings, facts, and body copy must use normal letter spacing. Existing uppercase micro-label tracking may remain where already established, but do not add negative tracking or viewport-scaled font sizes. Long state names, bill IDs, dates, source labels, and comparison sentences must wrap at 375px, 390px, 768px, and 1280px without clipping or overlapping.

Do not use auto-rotating or animated text for bill status, source footing, review dates, state names, or policy claims.
